Gene Set Clustering based on Functional annotation v1.1
----------------------------------------------------------------------------
GeneSCF v1.1 supports all organisms from KEGG and also from Gene Ontology. This update will have real-time funtional enrichment feature.
For new updates about GeneSCF, visit hosting page: https://github.com/genescf/GeneSCF
